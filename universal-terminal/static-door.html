The Static at my Door

During my sixth week at Base Camp, we were tasked with creating a choose-your-own-adventure game using Python with the knowledge we had acquired up to that point. The storyline of this project revolves around a sleep paralysis demon in your house, and your mission is to find a way to not be killed or taken away by this entity. Each decision you make carries consequences. The projects consequences are like "Secret Options", "Game Overs", "Good Endings", and "Bad Endings". Believe me or not, this project taught me how to improve my grammar and Python branching skills.
